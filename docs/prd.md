# 王者荣耀英雄互动小游戏需求文档

## 1. 游戏概述
### 1.1 游戏名称
王者荣耀英雄互动小游戏

### 1.2 游戏描述
一款基于王者荣耀英雄形象的互动展示类小游戏，玩家可以浏览不同英雄并体验英雄语音互动功能。

## 2. 游戏玩法
### 2.1 核心玩法
- 点击式导航浏览：通过点击按钮在不同界面间切换
- 英雄选择互动：点击英雄头像进入对应英雄详情页
- 语音互动体验：点击背景音效按钮播放英雄专属语音

### 2.2 操作方式
- 鼠标点击操作
- 界面按钮导航
- 音效按钮触发

### 2.3 游戏流程
1. 进入主界面，点击「开始游戏」按钮
2. 查看游戏提示界面，点击进入下一步
3. 进入英雄选择界面，点击任意英雄头像
4. 进入对应英雄详情页，可播放语音或返回选择界面

## 4. 技术功能
### 4.1 界面跳转
- 主界面 → 提示界面 → 英雄选择界面
- 英雄选择界面 ↔ 各英雄详情页
- 支持返回首页功能

### 4.2 音效系统
- 英雄语音播放功能
- 背景音效控制
- 音频文件加载和播放
- 预生成英雄台词内容并保存到后台，避免每次联网生成

### 4.3 声音选配功能
- 支持AI模仿语音生成
- 支持MP4音频文件上传配置
- 音频文件命名规范：kai.mp4、sunwukong.mp4、houyi.mp4、zhubajie.mp4、lvbu.mp4、guanyu.mp4、caiwenji.mp4等
- 音频文件存放在工程当前路径下
- 优先级：如存在对应英雄的MP4文件则播放该文件，否则使用AI生成语音
- 宏定义配置：在代码中设置英雄音频文件映射表，便于维护和扩展

### 4.4 图片资源
- 使用提供的游戏界面图片
- 支持英雄头像点击识别
- 界面背景图片适配
