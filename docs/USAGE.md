# 王者荣耀英雄互动小游戏 - 使用说明

## 游戏简介

这是一款基于王者荣耀英雄形象的互动展示类小游戏，玩家可以浏览不同英雄并体验英雄语音互动功能。

## 游戏流程

### 1. 主界面
- 打开游戏后，您将看到王者荣耀主题的欢迎界面
- 点击屏幕任意位置即可进入下一步

### 2. 提示界面
- 查看游戏提示信息
- 点击屏幕任意位置继续

### 3. 英雄选择界面
- 您将看到15个英雄头像
- 点击任意英雄头像进入该英雄的详情页
- 左上角有背景音效按钮（装饰功能）
- 目前开放的英雄：
  - **凯**（左上角第一个）- 绕口令挑战，使用成熟稳重的男性音色
  - **蔡文姬**（左上角第二个）- 表情包模仿，使用温柔可爱的女性音色

### 4. 英雄详情页
- 右上角有"返回首页"按钮，点击返回英雄选择界面
- 中间有"背景音效"按钮，点击后会：
  1. 显示"生成中..."加载状态
  2. 调用AI语音合成API生成英雄专属语音
  3. 自动播放生成的语音
- 每个英雄都有独特的语音内容

## 特色功能

### 🎤 AI语音合成
- 使用百度AI语音合成技术
- 实时生成英雄专属语音
- 高质量音频播放体验
- **凯**使用度小宇音色（成熟稳重的男性声音）
- **蔡文姬**使用度丫丫音色（温柔可爱的女性声音）

### 🎮 流畅交互
- 简洁直观的界面设计
- 流畅的页面切换动画
- 响应式布局，适配不同屏幕

### 🎨 王者荣耀风格
- 深蓝色+金色配色方案
- 游戏原画作为背景
- 符合王者荣耀视觉风格

## 技术特点

- **前端框架**: React + TypeScript
- **UI组件**: shadcn/ui + Tailwind CSS
- **路由管理**: React Router
- **语音合成**: 百度长文本在线合成API
- **响应式设计**: 适配PC和移动端

## 注意事项

1. 首次点击"背景音效"按钮时，需要等待几秒钟生成语音
2. 生成的语音会自动播放，请确保设备音量适中
3. 语音文件会缓存，第二次播放会更快
4. 部分英雄暂未开放，点击会提示"该英雄暂未开放"

## 英雄语音内容

### 凯 - 绕口令挑战
**音色特点**：成熟稳重的男性声音（度小宇音色，音调偏低）
```
四和十，十和四，十四和四十，四十和十四。
说好四和十得靠舌头和牙齿：谁说四十是"细席"，他的舌头没用力；
谁说十四是"适时"，他的舌头没伸直。
认真学，常练习，十四、四十、四十四。
```

### 蔡文姬 - 表情包模仿
**音色特点**：温柔可爱的女性声音（度丫丫音色，音调偏高）
```
请模仿穿边这个表情包。
老师拍照留念。
```
