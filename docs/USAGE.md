# 王者荣耀英雄互动小游戏 - 使用说明

## 游戏简介

这是一款基于王者荣耀英雄形象的互动展示类小游戏，玩家可以浏览不同英雄并体验英雄语音互动功能。

## 游戏流程

### 1. 主界面
- 打开游戏后，您将看到王者荣耀主题的欢迎界面
- 点击屏幕任意位置即可进入下一步

### 2. 提示界面
- 查看游戏提示信息
- 点击屏幕任意位置继续

### 3. 英雄选择界面
- 您将看到15个英雄头像
- 点击任意英雄头像进入该英雄的详情页
- 左上角有背景音效按钮（装饰功能）
- 目前开放的英雄：
  - **凯**（第1个）- 绕口令挑战，使用成熟稳重的男性音色
  - **蔡文姬**（第2个）- 表情包模仿，使用温柔可爱的女性音色
  - **孙悟空**（第3个）- 演唱《孤勇者》，使用活力四射的男性音色
  - **后羿**（第4个）- 走猫步挑战，使用沉稳的男性音色
  - **猪八戒**（第5个）- 四小天鹅舞蹈，使用憨厚的男性音色
  - **吕布**（第6个）- 跳舞挑战，使用霸气的男性音色
  - **关羽**（第7个）- 才艺表演，使用威严的男性音色

### 4. 英雄详情页
- 右上角有"返回首页"按钮，点击返回英雄选择界面
- 中间有"背景音效"按钮，点击后会：
  1. 显示"生成中..."加载状态
  2. 调用AI语音合成API生成英雄专属语音
  3. 自动播放生成的语音
- 每个英雄都有独特的语音内容

## 特色功能

### 🎤 AI语音合成
- 使用百度AI语音合成技术
- 实时生成英雄专属语音
- 高质量音频播放体验
- **凯**使用度小宇音色（成熟稳重的男性声音）
- **蔡文姬**使用度丫丫音色（温柔可爱的女性声音）

### 🎮 流畅交互
- 简洁直观的界面设计
- 流畅的页面切换动画
- 响应式布局，适配不同屏幕

### 🎨 王者荣耀风格
- 深蓝色+金色配色方案
- 游戏原画作为背景
- 符合王者荣耀视觉风格

## 技术特点

- **前端框架**: React + TypeScript
- **UI组件**: shadcn/ui + Tailwind CSS
- **路由管理**: React Router
- **语音合成**: 百度长文本在线合成API
- **响应式设计**: 适配PC和移动端

## 注意事项

1. 首次点击"背景音效"按钮时，需要等待几秒钟生成语音
2. 生成的语音会自动播放，请确保设备音量适中
3. **语音会自动缓存**，第二次播放会直接使用缓存，无需联网生成
4. 缓存保存在浏览器本地存储中，清除浏览器数据会删除缓存
5. 部分英雄暂未开放，点击会提示"该英雄暂未开放"

## 英雄语音内容

### 凯 - 绕口令挑战
**音色特点**：成熟稳重的男性声音（度小宇音色，音调偏低）
```
四和十，十和四，十四和四十，四十和十四。
说好四和十得靠舌头和牙齿：谁说四十是"细席"，他的舌头没用力；
谁说十四是"适时"，他的舌头没伸直。
认真学，常练习，十四、四十、四十四。
```

### 蔡文姬 - 表情包模仿
**音色特点**：温柔可爱的女性声音（度丫丫音色，音调偏高）
```
请模仿穿边这个表情包。
老师拍照留念。
```

### 孙悟空 - 演唱《孤勇者》
**音色特点**：活力四射的男性声音（度小宇音色，音调适中）
```
歌唱一首《孤勇者》
爱你孤身走暗巷，爱你不跪的模样
爱你对峙过绝望，不肯哭一场
爱你破烂的衣裳，却敢堵命运的枪
爱你和我那么像，缺口都一样
去吗？配吗？这褴褛的披风
战吗？战啊！以最卑微的梦
致那黑夜中的呜咽与怒吼，谁说站在光里的才算英雄
```

### 后羿 - 走猫步挑战
**音色特点**：沉稳的男性声音（度小宇音色，音调标准）
```
走猫步
绕全班过道用猫步走一圈，
停顿三次，并摆三个搞笑
的造型。
```

### 猪八戒 - 四小天鹅舞蹈
**音色特点**：憨厚的男性声音（度小宇音色，音调稍高）
```
四小天鹅
请找到你的小伙伴，并像
图片一样手挽手上台一起
跳一段四小天鹅。
```

### 吕布 - 跳舞挑战
**音色特点**：霸气的男性声音（度小宇音色，音调适中）
```
来跳舞吧
请拉上你两个好朋友
上台来表演这一段舞蹈。
```

### 关羽 - 才艺表演
**音色特点**：威严的男性声音（度小宇音色，音调偏低）
```
表演一段才艺
请上台表演一段你最拿手的才艺，
可以是唱歌、跳舞、讲笑话等。
```
