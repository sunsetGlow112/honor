# 更新日志

## [1.1.0] - 2025-11-02

### ✨ 新增功能

#### 新增5位英雄
- **孙悟空**: 演唱《孤勇者》挑战
- **后羿**: 走猫步挑战
- **猪八戒**: 四小天鹅舞蹈挑战
- **吕布**: 跳舞挑战
- **关羽**: 才艺表演挑战

#### 语音缓存系统
- 实现智能语音缓存功能
- 使用localStorage保存生成的语音URL
- 首次生成后自动缓存，第二次播放无需联网
- 添加缓存状态提示，用户可知是否使用缓存
- 显著提升用户体验，实现秒开播放

### 🎨 优化改进

#### 音色配置
- 为每个英雄配置专属音色参数
- 通过音调（pitch）区分不同角色性格
- 凯、关羽：音调偏低，威严成熟
- 孙悟空、吕布：音调适中，活力霸气
- 后羿：音调标准，沉稳可靠
- 猪八戒：音调稍高，憨厚可爱
- 蔡文姬：使用女性音色，温柔甜美

#### 用户体验
- 优化语音播放逻辑
- 添加详细的提示信息
- 改进错误处理机制

### 📚 文档更新
- 更新 USAGE.md，添加新英雄说明
- 创建 CACHE.md，详细说明缓存功能
- 创建 HEROES.md，提供英雄详细信息
- 更新所有文档中的英雄列表

### 🐛 修复问题
- 修复英雄选择界面的位置映射
- 优化API调用逻辑
- 改进音频播放控制

## [1.0.0] - 2025-11-02

### 🎉 首次发布

#### 核心功能
- 实现完整的游戏流程：主界面 → 提示界面 → 英雄选择 → 英雄详情
- 集成百度AI语音合成技术
- 实现英雄语音播放功能
- 响应式设计，支持PC和移动端

#### 初始英雄
- **凯**: 绕口令挑战
- **蔡文姬**: 表情包模仿

#### 技术栈
- React + TypeScript
- Tailwind CSS
- shadcn/ui 组件库
- React Router
- Axios

#### 设计风格
- 王者荣耀主题配色（深蓝色+金色）
- 使用游戏原画作为背景
- 精美的UI设计和交互效果
